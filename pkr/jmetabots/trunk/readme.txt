This projects contains java classes to connect to ai.pkr.metabots server.

It's in an incomplete state, just to be able to run simple PokerAcademy bots and FellOmen2.

It uses google protocol buffers. The protocol file is taken from ai.pkr.metabots.remote using
snv externals (without specific revision). The protocol file is compiled by tools\protoc.exe (2.2.0).
