Usage:
mvn archetype:create -DarchetypeGroupId=ai.bds -DarchetypeArtifactId=package-net -DarchetypeVersion=<current-version> -DgroupId=<your-group-id> -DartifactId=<your-artifact-id>

A new folder named after the artifact id will be created. It will contain a .NET solution 
(need to be renamed) and default directory structure.
